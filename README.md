# LibraryBook
Управление библиотекой книг

Описание функционала
1. Добавление книги: Пользователь вводит author, title и year, после чего книга добавляется в библиотеку с уникальным id и статусом “в наличии”.
 2. Удаление книги: Пользователь вводит id книги, которую нужно удалить.
 3. Поиск книги: Пользователь может искать книги по author, title или year.
 4. Отображение всех книг: Приложение выводит список всех книг с их id, author, title, year и status.
 5. Изменение статуса книги: Пользователь вводит id книги и новый статус (“в наличии” или “выдана”).

Каждая книга содержит следующие поля:
 • id (уникальный идентификатор, генерируется автоматически)
 • author (автор книги)
 • title (название книги)
 • year (год издания)
 • status (статус книги: “в наличии”, “выдана”)
